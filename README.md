# This Project is FrontEnd from IdeaPot

This is made with

- [Next.js](https://nextjs.org/)
- [Styled-Components](https://styled-components.com/)
- [TypeScript](https://www.typescriptlang.org/)

## Getting Stared

First, Install node_modules

```bash
  npm install
```

Second, Run Developer Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
